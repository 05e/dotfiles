# dMacOS Dotfiles

```sh
# download & install homebrew
curl -O https://raw.githubusercontent.com/05e/dotfiles/main/install-homebrew && sudo chmod 777 install-homebrew && ./install-homebrew
# download & install applications
curl -O https://raw.githubusercontent.com/05e/dotfiles/main/install-apps && sudo chmod 777 install-apps && ./install-apps
# download & install zsh config
curl -O https://raw.githubusercontent.com/05e/dotfiles/main/install-zsh && sudo chmod 777 install-zsh && ./install-zsh

# cleanup
rm install-zsh && rm install-apps && rm install-homebrew
```
