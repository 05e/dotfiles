# MacOS Dotfiles

```sh
# download & install homebrew
curl -O https://raw.githubusercontent.com/05e/dotfiles/main/macos/install-homebrew && sudo chmod 777 install-homebrew && ./install-homebrew
# download & install applications
curl -O https://raw.githubusercontent.com/05e/dotfiles/main/macos/install-apps && sudo chmod 777 install-apps && ./install-apps
# download & install zsh config
curl -O https://raw.githubusercontent.com/05e/dotfiles/main/macos/install-zsh && sudo chmod 777 install-zsh && ./install-zsh
# show nano install instructions
curl https://raw.githubusercontent.com/05e/dotfiles/main/macos/install-nano

# cleanup
rm install-zsh && rm install-apps && rm install-homebrew
```
